# Проблема с SMTP Mail.ru

## Проблема
Сервер не может подключиться к `smtp.mail.ru` на портах 465 и 587.
Ошибка: `Connection timed out` / `Network is unreachable`

## Причина
Провайдер хостинга (VDSina) блокирует исходящие SMTP соединения на портах 465 и 587.
Это стандартная практика для предотвращения спама.

## Решения

### Вариант 1: Использовать альтернативный SMTP сервис
- **SendGrid** - бесплатный тариф до 100 писем/день
- **Mailgun** - бесплатный тариф до 5000 писем/месяц
- **Amazon SES** - очень дешево, надежно
- **Yandex SMTP** - если есть аккаунт Yandex

### Вариант 2: Связаться с провайдером
Запросить разблокировку портов 465/587 для исходящих SMTP соединений.

### Вариант 3: Использовать прокси/VPN
Настроить прокси-сервер для SMTP соединений (сложно).

### Вариант 4: Использовать API Mail.ru
Если Mail.ru предоставляет API для отправки писем (нужно проверить).

## Рекомендация
Использовать **SendGrid** или **Mailgun** для отправки писем через их API/SMTP.
Эти сервисы обычно не блокируются провайдерами.
