# Замена alert() на систему уведомлений

## Выполнено

### 1. Создана система уведомлений (`notification.js`)
- Красивые toast-уведомления вместо `alert()`
- Поддержка типов: success, error, warning, info
- Автоматическое исчезновение через 5-6 секунд
- Анимация появления/исчезновения
- Иконки Lucide для каждого типа
- Адаптивный дизайн для мобильных устройств

### 2. Создана система модальных окон (`modal.js`)
- Красивые модальные окна вместо `confirm()` и `prompt()`
- Поддержка типов: info, confirm, prompt, warning, error
- Поддержка HTML контента
- Поддержка разных типов input (text, number)
- Закрытие по Escape и клику на backdrop
- Анимация появления/исчезновения

### 4. Функции уведомлений

```javascript
// Простые уведомления
notifySuccess(message, duration = 5000)
notifyError(message, duration = 6000)
notifyWarning(message, duration = 6000)
notifyInfo(message, duration = 5000)

// Модальные окна
confirmModal(message, title = 'Подтверждение')
promptModal(message, defaultValue = '', placeholder = '', inputType = 'text')
showModal(options)
```

## Примеры использования

```javascript
// Успешное уведомление
notifySuccess('Профиль сохранен!');

// Ошибка
notifyError('Ошибка сохранения профиля');

// Предупреждение
notifyWarning('Недостаточно средств для вывода');

// Подтверждение
const confirmed = await confirmModal('Вы уверены, что хотите выйти?', 'Выход из аккаунта');
if (confirmed) {
    // Действие
}

// Ввод данных
const amount = await promptModal('Введите сумму', '', 'Введите сумму для вывода', 'number');
```

## Особенности

1. **Уведомления**:
   - Появляются справа вверху
   - Автоматически исчезают через 5-6 секунд
   - Можно закрыть вручную
   - Поддержка многострочных сообщений
   - Иконки Lucide для каждого типа

2. **Модальные окна**:
   - Центрированы на экране
   - Закрываются по Escape или клику на backdrop
   - Поддержка HTML контента
   - Поддержка разных типов input
   - Анимация появления/исчезновения

3. **Адаптивность**:
   - Уведомления адаптируются под размер экрана
   - Модальные окна адаптируются под мобильные устройства
   - Поддержка touch-событий
